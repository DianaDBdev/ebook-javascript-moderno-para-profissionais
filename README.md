## 📖 Sobre o livro

**JavaScript Moderno para Profissionais** é um guia prático para desenvolvedores com experiência em JavaScript que querem dominar o ecossistema moderno — sem começar do zero.

Em 14 capítulos, você vai dominar ESM, TypeScript, Vite 8, async/await avançado, arquitetura modular, testes com Vitest, WebSockets, SSR com React Server Components, deploy com edge computing, migração de projetos legados e muito mais.

## ⚡ Como usar este repositório

### Pré-requisitos

- [Node.js 20+](https://nodejs.org/)
- [Git](https://git-scm.com/)
- Editor de código (recomendado: [VS Code](https://code.visualstudio.com/))

### Instalação

```bash
# 1. Clone o repositório
git clone https://github.com/DianaDBdev/javascript-moderno-profissionais.git

# 2. Entre na pasta do capítulo que está estudando
cd javascript-moderno-profissionais/capitulo-01-esm

# 3. Instale as dependências (quando houver)
npm install

# 4. Execute o exemplo
npm start
# ou
node index.js
```

### Cada capítulo é independente

Você não precisa instalar tudo de uma vez. Cada pasta tem seu próprio `package.json` quando necessário. Vá direto ao capítulo que está lendo.
