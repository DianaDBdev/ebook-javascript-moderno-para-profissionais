# Capítulo 10 — Real-Time com WebSockets

> Exemplos de código do **Capítulo 10** do livro *JavaScript Moderno para Profissionais*.

## Arquivos principais

```
ws-server.js (servidor), ws-client.js (cliente com reconexão)
```

## ▶️ Como executar

```bash
cd capitulo-10-websockets
node <arquivo>.js
# Para TypeScript:
npx tsx <arquivo>.ts
```

## 💡 Conceitos do capítulo

WebSocket server/client, reconexão com backoff exponencial, heartbeat, broadcast, Server-Sent Events
